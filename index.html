Títulos em HTML:

Voltar

As tags h1 até h6 são usadas para criar títulos e subtítulos dentro de uma página HTML.

O h1 representa o título principal, sendo o mais importante e com maior destaque visual.

Conforme o número aumenta, de h2 até h6, a importância e o tamanho do texto diminuem.

Essas tags ajudam a organizar o conteúdo, melhorar a leitura e facilitar a estrutura da página.

Elas também são importantes para SEO, pois indicam a hierarquia das informações.

Listas Ordenadas e Não Ordenadas:

Voltar

As listas em HTML são usadas para organizar informações em forma de itens.

A lista ordenada, representada pela tag ol, cria uma lista numerada, onde cada item recebe um número automaticamente.

A lista não ordenada, representada pela tag ul, cria uma lista com marcadores (bolinhas).

Tanto no ol quanto no ul, cada item da lista é definido pela tag li.

Esse tipo de estrutura ajuda a deixar as informações mais claras e organizadas.

Link e Navegação:

Voltar

A tag a é conhecida como âncora e serve para criar links em uma página HTML.

Com ela, é possível direcionar o usuário para outro site, outra página do mesmo site ou até partes específicas da página atual.

Para isso, usamos o atributo href, onde colocamos o endereço do link.

Por exemplo:a href="https://example.com" Clique aqui /a

Essa tag é fundamental para a navegação na web.
